Multiplayer Boggle Game

My Favorite Project

This project was for software engineering II. It was created in a group of 4. It required that it be able to be played by two people, connected to a database, have accounts, and used react.js

It was a very challenging project for the whole class. None of us had learned React yet so it required that we learn all we can outside of class. Getting the game logic set up and the individual logins working went by smoothly. Getting the two players to communicate was the challenging part.

Our solution was to use the database to create a lobby. Each move made by the player would be stored to the database. The game would check every 500ms for updates to the database and would update each players game accordingly.
