This project was for my capstone course. It was a tenure project for one of the professors. It would take data from two algorithms and compare memory usage, times, etc between the two. We did not have the C++ backend for the algorithms so we used dummy data and would return the info that was selected. We used mongoDB and react.js in this project.


I handled most of the CSS and layouts as well as the majority of the form controls and their handler functions.
